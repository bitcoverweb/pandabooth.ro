# Dashboard Panda Booth 📊

Un dashboard modern și frumos, similar cu imaginea atașată.

## 🎯 Caracteristici:

✅ **Sidebar navigare** - cu meniu principal și setări
✅ **Header cu acțiuni** - Upgrade, notificații, etc.
✅ **Statistici frumoase** - Efficiency, Tasks, Online time
✅ **Grafice interactive** - Performance review și Team performance
✅ **Need review section** - Lista oamenilor care au nevoie de review
✅ **Departments cards** - UI designers, Accounting, Social Media, SEO
✅ **Design responsive** - Funcționează pe desktop și mobile
✅ **Animații smooth** - Efecte hover și tranziții

## 📁 Fișiere:

- **index.html** - Structura HTML
- **style.css** - Styling CSS (modern, clean)
- **script.js** - JavaScript pentru grafice (Chart.js)

## 🚀 Cum să deschizi:

```
http://localhost:8000/dashboard/
```

## 📊 Grafice incluse:

1. **Performance Review** - Doughnut chart cu status-uri
2. **Team Performance** - Line chart cu trend-uri de la Winter la Autumn

## 🎨 Coloruri:

- Primary: #667eea (albastru)
- Secondary: #764ba2 (mov)
- Success: #6bcf7f (verde)
- Danger: #ff6b6b (roșu)

## ✨ Funcționalități interactive:

- Click pe nav items → active state
- Click pe edit icon → mesaj
- Click pe "Manage personal" → mesaj
- Click pe "Add team" → mesaj
- Click pe bell icon → notificări
- Click pe "Upgrade plan" → mesaj

## 📱 Responsive Design:

- Desktop: 2 coloane layout
- Tablet/Mobile: 1 coloană layout
- Sidebar ascuns pe mobile
